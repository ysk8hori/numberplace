# Numberplace

ナンプレのゲームを遊べるアプリです。 React と vite を使い、 PWA として構築する予定です。バックエンドとの通信は不要で、静的なフロントエンドアプリケーションとして動作します。

子供でも遊べるよう優しい作りにします。

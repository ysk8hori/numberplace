ナンプレのゲームを遊べるアプリです。 バックエンドとの通信は不要で、静的なフロントエンドアプリケーションとして動作します。PWA としてインストールも可能。

子供でも遊べるよう優しい作りにします。

AWS にデプロイ済みです。
https://numberp.net
